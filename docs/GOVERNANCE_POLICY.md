# سياسة الحوكمة (مُطبقة) 

## حزمة البرامج المُقفلة 
- Next.js (بدون بديل) 
- Prisma (بدون ORM إضافي) 
- PostgreSQL 
- Tailwind + shadcn/ui (بدون إطار عمل واجهة مستخدم ثانٍ) 
- نشر Vercel 

## ممنوع بدون موافقة صريحة (CR) 
- استبدال/إضافة ORM (Drizzle/TypeORM/إلخ) 
- إضافة أنظمة مصادقة بديلة 
- إدخال مديري حالة إضافيين (Redux/Zustand/MobX) إذا كان أحدهما موجودًا بالفعل 
- تغيير منصة النشر 
- عمليات إعادة هيكلة كبيرة (أكثر من 300 سطر من التعليمات البرمجية) ما لم يُطلب ذلك 
- أي تغييرات في المناطق المحمية: 
  /prisma 
  /src/lib/auth 
  /src/lib/security 
  /src/lib/ledger 
  /src/app/admin-core 

## التغييرات المسموح بها 
- إصلاحات الأخطاء ضمن النطاق 
- ميزات صغيرة ضمن النطاق 
- تصحيحات أمنية مع دليل (اختبار/سجل/إثبات المفهوم) 

## تعريف الاكتمال 
- CI أخضر 
- تم إكمال نموذج طلب السحب 
- اختلاف طفيف 
- تم الالتزام بقواعد الترحيل
