# مواصفات منتج Recycle24

## 1. نظرة عامة
- اسم المنتج: Recycle24
- الوصف: منصة متكاملة لإعادة التدوير
- الرؤية: المنصة الرائدة في الشرق الأوسط

## 2. المستخدمون
- تاجر (TRADER)
- مشتر (BUYER)
- سائق (DRIVER)
- مسؤول (ADMIN)

## 3. الميزات الرئيسية
- المصادقة والحساب
- المزادات (Auctions)
- السوق (Market)
- النقل (Transport)
- المحفظة (Wallet)
- GSCC للأمان

## 4. التقنية
- Next.js 16
- PostgreSQL (Neon)
- Prisma ORM
- TypeScript
